# 💻 isl-Tech Website

**isl-Tech** — Solomon Islands Technology Solutions

A modern, responsive business website for isl-Tech, a Solomon Islands-based tech venture offering app development, digital products, and consulting services.

## 🚀 Features

- 🌗 **Dark/Light mode** — auto-detects system preference, user can toggle
- 📱 **Fully responsive** — mobile-first design, works on all devices
- 📲 **Apps page** — browse and download Android/iOS/web apps
- 🛍️ **Products page** — digital products, software, and services
- 📬 **Contact form** — enquiry form with subject routing
- ⚡ **Lightweight** — pure HTML/CSS/JS, no frameworks, fast loading
- 🎨 **Modern design** — gradient accents, smooth animations, clean typography

## 🗂️ Site Structure

```
isl-tech.com/
├── index.html          # Home page
├── apps.html           # App downloads & testing
├── products.html       # Shop / products & services
├── about.html          # About the founder & mission
├── contact.html        # Contact form & details
├── blog/               # (Coming soon) Blog & updates
├── assets/
│   ├── css/style.css   # Main stylesheet
│   └── js/main.js      # Theme toggle, nav, animations
└── README.md
```

## 🛠️ Tech Stack

| Layer | Technology |
|-------|-----------|
| **Markup** | HTML5 |
| **Styling** | CSS3 (Custom properties, Grid, Flexbox) |
| **Scripting** | Vanilla JavaScript (ES6+) |
| **Icons** | Emoji (no icon library needed) |
| **Fonts** | System font stack (fast loading) |

## 🌐 Hosting

Recommended hosting platforms:
- **GitHub Pages** — Free, easy deployment
- **Netlify** — Free tier, form handling built-in
- **Vercel** — Excellent for static sites

## 🔧 Customization

1. **Logo**: Replace the `brand-icon` div in each navbar with your logo image
2. **Content**: Update apps, products, and about text in the HTML files
3. **Links**: Update social links, download URLs, and contact info
4. **Domain**: Point `isl-tech.com` to your hosting provider
